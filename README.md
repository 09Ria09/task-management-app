# Talio

## Description of project

Talio is a versatile task management and collaboration application designed to streamline team workflows and project organization. Utilizing a client-server architecture, Talio enables multiple users to connect to a server and work together on a shared board in real-time.

## Features

- **Flexible Server Connection**: Users can easily connect to and switch between Talio servers to manage tasks.
- **Multiple Boards**: Create, join, and navigate between multiple boards for better organization of various projects or categories.
- **Task and List Management**: Add, edit, remove, and reorder tasks within lists, and manage lists to categorize tasks effectively.
- **Intuitive Drag and Drop Interface**: Rearrange tasks on the board effortlessly with a user-friendly drag and drop system.
- **Detailed Task Cards**: Enhance task organization by adding descriptions, subtasks, and other relevant details to task cards.
- **Task Tagging System**: Assign and manage tags for tasks, creating a flexible and consistent classification method.
- **Customizable Appearance**: Personalize board, list, and task appearances using color schemes and presets for a unique user experience.
- **Keyboard Shortcuts**: Boost productivity with a variety of keyboard shortcuts designed for efficient task navigation and management.
- **Real-time Auto Synchronization**: Instantly synchronize changes across all connected clients, ensuring everyone stays up-to-date without manual refreshing.

## How to run it

To begin using Talio, follow these steps:

- Clone the repository or download the source code. Make sure you have JavaFX installed on your computer  
        you can do that by using these commands in your terminal

    ```sh
    git clone git@gitlab.ewi.tudelft.nl:cse1105/2022-2023/teams/oopp-team-29.git
    cd oopp-team-29
    ```
- Open the project in your preferred Java IDE.    
- Compile and run the server application. (optional)  
you can do this by using the command
```sh
./gradlew bootrun
```
 in the terminal or by just running the server  
- Compile and run the client application, then connect to your preferred server using its IP address.  
you can do this by using the command
```sh
./gradlew run
```

## Usage

After connecting to a Talio server, users can create boards, lists, and tasks to organize their work effectively. Additional features, such as card details, tags, and customization options, can be accessed through the user interface.
For a more detailed explanation of each feature and its functionality, please refer to the product pitch of Talio.
## How to contribute to it

We welcome contributions to Talio. To contribute, just   
1. Open an issue in the project's issue tracker and describe the feature, bugfix, or improvement you want to work on.
2. Create a new branch from the `main` branch, and name it according to the issue you are working on.
3. Start coding your solution while making sure to document and to test your code as needed.
4. After completing your work, pull/merge from `main` to your local branch. Resolve any merge conflicts that arise.
5. Finally, create a merge request from your local branch to the `main` branch. Request a review from at least two team members, and incorporate any feedback before merging the changes.
